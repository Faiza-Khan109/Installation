Complete Windows Python Development Setup

This guide walks through the full installation and configuration process for setting up a Python development environment on Windows using Python, Anaconda, and Jupyter Notebook.

What You Will Install

•	Python (Programming Language)

•	Anaconda (Scientific Distribution)

•	Jupyter Notebook (Interactive Coding Environment)

Step 1: Install Python

1.	Visit https://www.python.org/downloads/	

2.	Download the latest stable version.

3.	During installation:

o	Select “Install for all users”

o	Check “Add Python to PATH”

4.	Click “Install Now”

5.	Disable path length limit when prompted.

This ensures Python is accessible system-wide.

Step 2: Install Anaconda

1.	Download from https://www.anaconda.com/download

2.	Click Download (Skip Registration if needed)

3.	Accept License Agreement

4.	Choose installation type:

o	“Just Me” (recommended)

5.	Choose install directory (avoid spaces)

6.	Complete installation

Anaconda includes:

•	Jupyter Notebook

•	Conda Package Manager

•	Pre-installed scientific libraries

Step 3: Create a Jupyter Notebook

1.	Open Anaconda Navigator

2.	Launch Jupyter Notebook

3.	Select your working directory

4.	Click “New” → “Python 3”

5.	A new .ipynb notebook file opens

6.	Start writing and executing Python code

Why This Setup?

This configuration provides:

•	A stable Python environment

•	Easy package management

•	Interactive coding support

•	Ideal setup for data science & ML labs
